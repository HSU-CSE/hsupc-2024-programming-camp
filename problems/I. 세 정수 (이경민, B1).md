# I. 세 정수

| 시간 제한 | 메모리 제한 |
| --- | --- |
| 1초 | 256MB |

## 문제
절대값이 $N$보다 작거나 같은 세 정수 $a, \\ b, \\ c$에 대해

$(a + b)^2 = (a + b + c)^2$

위 식을 만족하는 $(a, \\ b, \\ c)$ 정수 쌍의 개수를 구하라.



## 입력
첫째 줄에 $N \\ (1 \leq N \leq 4000)$이 주어진다.

## 출력
식을 만족하는 세 정수 쌍의 개수를 출력하라.

## 예제 입력

```
10
```

## 예제 출력

```
631
```
