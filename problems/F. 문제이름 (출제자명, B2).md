# A. 상상부기 언어

| 시간 제한 | 메모리 제한 |
| --- | --- |
| 2초 | 1024MB |

## 문제
상상부기와 얘기하던 도중 갑자기 상상부기는 자신이 원래 사용하던 언어를 구사하게 된다.


## 입력
상상부기의 언어가 입력으로 들어오게 된다.

## 출력
상상부기의 언어를 해석하여 정답을 출력해보자.

## 예제 입력

```
/e25<+e9+5;2
```

## 예제 출력

```
I LOVE SEOUL
```


## Hint 
```

```

## feat
```
꾸륵 꾸륵 이게 맞나요? 부기부기!
```
